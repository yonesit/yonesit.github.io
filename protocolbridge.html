Protocolbridge

Protocolbridge ist eine IIoT-Middleware zur Verbindung von Modbus, OPC UA und MQTT über eine gemeinsame REST API und ein Echtzeit-Dashboard.

PROJEKTÜBERBLICK

Protokolle verständlich verbinden

Industrielle Geräte verwenden häufig unterschiedliche Kommunikationsprotokolle. Protocolbridge führt diese Daten in einer gemeinsamen Architektur zusammen, normalisiert sie und stellt sie für Anwendungen und Überwachungssysteme bereit.

  • Verarbeitung von Modbus-, OPC-UA- und MQTT-Daten
  • gemeinsame REST API für normalisierte Gerätedaten
  • WebSocket-Kommunikation für Echtzeit-Updates
  • Speicherung und Auswertung von Zeitreihendaten
  • Dashboard für Werte, Zustände und Alarme
